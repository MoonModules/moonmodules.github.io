## Getting started

Not sure which product to use? The [What to use](../products/index.md#what-to-use) table in the Products section gives a quick answer based on your setup — number of LEDs, fixture type, and use case.

For hardware, the [Buying guide](buying-guide.md) lists microcontrollers, shields, and peripherals we have tested in real projects.

**Flashing firmware** — the easiest way to get started is the [WLED web installer](https://install.wled.me/) (Chrome or Edge, not Safari). For WLED-MM or MoonLight, download the latest `.bin` from the GitHub releases and flash with the [ESP flasher by srg74](https://github.com/srg74/WLED-wemos-shield/tree/master/resources/Firmware/WLED_%20ESP_Flasher).

Community-maintained resources:

- [Wladi's WLED FAQ](https://wled-faq.github.io) — unofficial documentation in German and English
- [Wladi's WLED power calculator](https://wled-calculator.github.io) — calculate power supply requirements

**LED reference — [QuinLED.info](https://quinled.info)**

QuinLED is an independent resource covering everything about addressable and analog LED systems. Useful pages:

- [How much power do addressable LEDs use?](https://quinled.info/2020/03/12/digital-led-power-usage/) — real-world measured power values per LED type; more reliable than the "60 mA per LED" rule of thumb
- [What LED strip to buy](https://quinled.info/2019/06/03/what-digital-5v-12v-rgbw-led-strip-to-buy/) — strip selection guide covering 5 V, 12 V, and 24 V options
- [Power injection guide](https://quinled.info/the-ultimate-led-strip-power-injection-guide/) — how to inject power along a strip to avoid voltage drop on longer runs
- [Tools and equipment](https://quinled.info/2018/10/01/tools-and-equipment/) — recommended tools for building and testing LED installations
