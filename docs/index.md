# MoonModules

MoonModules is a collective of light artists, hardware and software developers. We build open source lighting solutions based on ESP32 microcontrollers.

Our work is used by makers, artists and engineers to drive everything from small hobby installations to large stage setups. All software is released under GPL-V3 and EUPL-v3 licenses.
